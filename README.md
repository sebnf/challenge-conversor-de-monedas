# Conversor de Monedas - Challenge Alura
Este es un conversor de monedas interactivo desarrollado en **Java**, que utiliza la **ExchangeRate-API** para obtener tasas de cambio en tiempo real.

## Funcionalidades
- Conversión entre USD, CLP, ARS, BRL y EUR.
- Consulta de tasas en tiempo real mediante API.
- Historial de conversiones realizadas durante la sesión.
- Interfaz amigable por consola.
